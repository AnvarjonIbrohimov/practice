  🥇 **Asosiy tushunchalar**
    - Notification → foydalanuvchiga keladigan bildirishnoma
    - description → batafsil izoh
    - title → sarlavha (nomi)
    - content → asosiy matn

  🥈 **Odam va bog‘liq narsalar**
    - receiver → xabarni qabul qiluvchi odam
    - rank → daraja (masalan: admin, user)

  🥉** Biznes / loyiha bilan bog‘liq**
    - property → mulk (uy, ofis va h.k.)
    - property desc → mulk haqida ma’lumot
    - rent → ijara (ijaraga berish)
    - square → maydon (m²)

  📂 **Turlash / kategoriyalar**
    - category → tur (masalan: uy, ofis)
    - article → maqola yoki post

  💈 **Boshqa**
    - barber → sartarosh (agar barber app bo‘lsa)
    - warningAuthType → autentifikatsiya ogohlantirish turi

  🧠 **Diagramma va tizim**
    - flowchart → jarayon diagrammasi (qadamlar ketma-ketligi)
    - ER Model (Entity Relationship) → database tuzilmasi diagrammasi

  💾 **Datatype (ma’lumot turlari)**
    - double → o‘nlik son (masalan: 12.5)
    - ARR (Array) → massiv (ro‘yxat)
    - NN (Not Null) → bo‘sh bo‘lishi mumkin emas
