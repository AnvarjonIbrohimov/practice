**GIT version control tizimi kompyuterimizga ornratamiz**


**Git ozi nima va unda qanday komandalar mavjud?**
  * git --version *  => orqali qaysi versiyada tekshirish 
  * git config --global user.name * =>kim ishlatyatganini tekshirish
  * git config --global user.name "Alex"* shunday qilib ozimiz hohlagan ismni kiritishimiz mumkin
  * git config --global user.email * emailni tekshirish 
  * git config --global --list* umumiy configurationni tekshirish uchun
  * git commit -m "..." * gitga joylashdan oldin commit yozish
  * git status * bu hozirda joylashinishi kerak bolgan malumot bormi shuni tekshiradi
  * git log --oneline * bu hozirgacha yuklangan commitlar va datalarni korsatadi
